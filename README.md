# 🏒 Mighty Ducks Hockey League - Mobile App

## 📄 Project Description 

MDHL Mobile App is a mobile-friendly application to provide information about Mighty Ducks children's hockey league games.

The requirement was to provide a user-friendly website accessible from phones or tablets that includes information such as game and event dates, maps to help people get there, information about the teams playing each other and their players, and also keeps track of completed game results.

The option to register and participate by leaving comments about the matches was also included.
